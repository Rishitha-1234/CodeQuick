Floyd-Warshall Algorithm:

The Floyd-Warshall algorithm is used to find shortest paths in a weighted graph with positive or negative edge weights (but no negative cycles). It computes the shortest path between all pairs of vertices by iteratively improving an estimate of the path between any two vertices through a third vertex . It does this by updating a distance matrix where each entry dist[i][j] represents the shortest distance from vertex i to vertex j. The algorithm runs in O(V^3) time complexity, making it suitable for graphs with up to a few hundred vertices due to its cubic time complexity. It is especially useful for finding shortest paths in dense graphs or when the exact source and destination are not known in advance.

                void floydWarshall(int graph[][V]) {
                    int dist[V][V], i, j, k;
                
                    for (i = 0; i < V; i++)
                        for (j = 0; j < V; j++)
                            dist[i][j] = graph[i][j];
                
                    for (k = 0; k < V; k++) {
                        for (i = 0; i < V; i++) {
                            for (j = 0; j < V; j++) {
                                if (dist[i][k] + dist[k][j] < dist[i][j])
                                    dist[i][j] = dist[i][k] + dist[k][j];
                            }
                        }
                    }
                
                    printSolution(dist);
                }

              V = 4
              
              # Define Infinite as a large enough value. This value will be used
              # for vertices not connected to each other
              INF = float('inf')
              
              # Algorithm implementation
              def floydWarshall(graph):
                  dist = list(map(lambda i: list(map(lambda j: j, i)), graph))
              
                  # Adding vertices individually
                  for k in range(V):
                      for i in range(V):
                          for j in range(V):
                              dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j])
                  
                  printSolution(dist)
